# Presupuesto Rápido

Aplicación web estática para generar presupuestos con un formato similar al de la imagen proporcionada.

## Uso rápido

1. Abrí `index.html` en el navegador.
2. Editá los datos del comercio y cliente directamente sobre el documento (campos `contenteditable`).
3. Modificá productos en la tabla: código, descripción, cantidad y precio.
4. Agregá o eliminá filas según necesidad.
5. Ajustá IVA y usá **Imprimir / Guardar PDF** para exportar.

## Características

- Formato estilo hoja A4.
- Cálculo automático de subtotal, impuestos y total.
- Edición rápida en línea sin backend.
- Estilos de impresión para generar PDF limpio.
